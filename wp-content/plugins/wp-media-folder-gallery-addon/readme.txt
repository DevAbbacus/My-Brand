WP media Folder Gallery Addon is a WordPress plugin that enhance the WordPress media manager by adding galleries features.

== Description ==

The WP Media Folder Gallery Addon brings image album management to WordPress.
Besides the WP Media Folder plugin's native features you'll get the possibility to create image galleries from existing folders.
The Addon brings 7 themes for your galleries, fully fledged gallery management while creating content (posts, pages, any text editor...) and multi-level galleries with image tag filtering.
